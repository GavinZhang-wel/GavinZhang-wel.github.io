- 博客与技术文章
- 项目与作品集
- 技术分享与笔记

## 技术栈
- 静态站点：GitHub Pages
- 构建：可以使用纯 HTML/CSS/JS，或使用 Hexo/ Hugo / Jekyll 等静态站点生成器
- 其他：可部署自定义域名与 HTTPS

## 本地预览（示例）
1. 克隆仓库：
   git clone https://github.com/GavinZhang-wel/GavinZhang-wel.github.io.git
2. 进入目录并在本地启动（取决于站点生成器）：
   - 纯静态文件：直接在浏览器中打开 index.html
   - Jekyll：
     gem install bundler jekyll
     bundle exec jekyll serve
   - Hexo：
     npm install
     hexo s
3. 打开浏览器访问：http://localhost:4000（或生成器默认端口）
